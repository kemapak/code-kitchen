HTML Session and Local Storage

Local storage is persistent, unless it is cleared programmatically or by the user clearing their browser cache.

Session storage is not persistent and exists during the session. It could also be cleared programmatically or by the user if the user closes the browser window and end the session or clears their browser cache.

Both local and session storage should be retrieved on a page reload, but if you close the browser window and reopen it session storage will be gone.

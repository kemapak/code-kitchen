Button Test

This file test the behavior of both button tag and input type=button tag. Open your console to see the behavior.

<button>Button</button> tag by default submits a form if inside a <form> tag. It could also take property type which can have values "submit", "reset" or "button"

<button>Button</button> tag also capable of having other tags inside. You can put images, icons or text inside button tags.

<input type="text"/> tag on the other hand does not submit a form can be used for invoking any action inside a form.

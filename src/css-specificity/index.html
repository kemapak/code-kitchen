CSS Specificity

This code show how the CSS specificity works and how to override it.

Remember inline style always takes precedence and override any class, or ID.

Always remember never rely on the order of the classes declared and !important declarations. Things could and will change without your knowledge by other developers and break the look and feel of your applications.

A pseudo-element is used to show the colors. For example: .blue::after {content: 'I am blue!';}. This is not show UI, look at the source of this file.
